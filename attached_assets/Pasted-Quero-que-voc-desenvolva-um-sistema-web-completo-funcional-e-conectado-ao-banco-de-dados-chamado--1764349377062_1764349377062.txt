Quero que você desenvolva um sistema web completo, funcional e conectado ao banco de dados, chamado BO.Militar, baseado exatamente nas telas apresentadas, incluindo login, cadastro, painel do cidadão, painel do atendente, registro de ocorrências, visualização, chat e atualização de status.

O sistema deve possuir front-end, back-end e banco de dados plenamente integrados.

Utilizar:

Front-end moderno (React ou equivalente)

Back-end em API REST (Node.js / Express ou similar)

Banco de dados relacional (MySQL, PostgreSQL ou SQLite)

Autenticação com senha criptografada e JWT

Separação por perfil de acesso: CIDADÃO e ATENDENTE (PM / Bombeiros)
